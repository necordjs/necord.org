# Финансирование {#funding}

Necord - это проект с открытым исходным кодом, его постоянное развитие стало возможным благодаря поддержке сообщества.
Эта библиотека - результат долгого пути, полного бессонных ночей, работы в нерабочее время и напряженных выходных.

## How can you help? {#how-i-can-help}

Будем благодарны за вашу финансовую поддержку Necord!
