# Financiamento {#funding}

Necord é um projeto open source com o seu desenvolvimento contínuo que é possível graças ao apoio da comunidade.
Esta biblioteca é um resultado de uma estrada longa, cheia de noites sem dormir, trabalhando após horas, e finais de semana ocupados.

## Como você pode ajudar? {#how-i-can-help}

Se você deseja apoiar financeiramente o Necord, agradeço!
