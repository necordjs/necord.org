# Financement {#funding}

Necord est un projet open source dont le développement continu est rendu possible grâce au soutien de la communauté.
Cette bibliothèque est le fruit d'un long parcours, fait de nuits blanches, d'heures supplémentaires et de week-ends bien remplis.

## Comment pouvez-vous aider ? {#how-i-can-help}

Si vous souhaitez soutenir Necord financièrement, merci à vous !
